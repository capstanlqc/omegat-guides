## Вставка терминов из глоссария

Если в текущем сегменте текст оригинала содержит термин, записанный в глоссарии, то такой термин будет выделяться _синим подчеркиванием_{ .term }.
Предлагаемый перевод термина будет показан в области **Glossary** (Глоссарий) в правой части окна программы.

![](../_img/20_glossary_term.jpg)

В OmegaT есть функция **предикативного ввода**: когда вы начинаете вводить первые буквы перевода термина, содержащегося в глоссарии, модуль автозавершения предложит весь термин. Для принятия предложенного варианта нажмите клавишу ++enter++.

![](../_img/21_predictive_typing.jpg)
